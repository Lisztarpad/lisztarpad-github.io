Coinciding Checker

When a processor clicks into a specific case, the system automatically checks whether the task is assigned to them, preventing collisions with colleagues who might be assigned the same task.

Piros Alarm

When a case involves special circumstances such as VAT or batch processing, this program will highlight critical areas in red to draw attention.

Quick GL

The system has preset many commonly used GL codes in the form of a secondary menu, and the GL data list is customizable. Users can quickly select a category to retrieve the required GL value, and the program will automatically copy it to the clipboard.

Quick Select

When a certain portion of text is selected in the tool (usually achieved by double-clicking), the selected content will be automatically copied to the clipboard.

Amount Unifier

When users copy amounts from invoices, irregular formats often occur, such as using commas for decimals and dots for thousands separators. This feature actively monitors numeric text in the clipboard and automatically converts it to the standard currency format accepted by SAP.

VAT Assistant

For cases involving VAT, the program will automatically generate a tax-related assignment code in the format of "F + country code + 1". In the final step of entering FIC, it will also copy the assignment code and FIC number simultaneously

Super Copy

This is the most powerful and practical feature of the program. For direct invoices, LFP needs to provide FC, Line, and corresponding amounts, often involving multiple FC entries. This feature can automatically calculate the total FC provided by LFP, integrate the selected GL, and generate a funding source table with one click, which can be copied into SAP. This enables balancing with a single click and brings the total to zero.

Fast Return

When further collaboration is needed from LFP, the program will automatically generate template messages for returns. These templates include salutations, closing remarks, and context-specific content based on scenarios such as duplication, insufficient funding, missing approval, house bank mismatch, etc. Users can also customize templates for other scenarios. This feature supports a one-click jump to the Teams chat interface with LFP and copies the pre-generated text for easy communication.

Running Platform:
This program runs on Microsoft Windows operating systems and supports the majority of mainstream versions (Windows 7 and above, compatible with both 32-bit and 64-bit).

Framework required:
This program is developed using Microsoft Visual Studio as the Integrated Development Environment (IDE) and is written in the C# programming language. The runtime environment requires the operating system to have .NET Framework 6.0 or a higher version installed. Normally, .NET Framework is pre-installed in Windows operating systems. If it's not installed, the program will automatically prompt and initiate the download and installation from the Microsoft official website.

Execution:
Due to UNICEF's security measures, this program cannot run directly from a local disk. The program needs to be stored in the cloud (such as OneDrive) before it can be executed.
